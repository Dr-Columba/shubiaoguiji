# 鼠标轨迹

一个面向 Windows 11 的鼠标指示与拖尾工具，基于 C++/Win32 实现，强调低资源占用与稳定运行。

## 功能特性

- 透明穿透置顶叠加层（不阻挡鼠标点击）
- 鼠标圆点指示 + 轨迹拖尾
- 托盘菜单配置
- 颜色模式：自定义颜色 / 彩虹色
- 粗细档位：粗 / 中 / 细（50 / 20 / 7）
- 拖尾残留：明显 / 适中 / 轻微（0.5 / 1.0 / 2.0）
- 透明度：高 / 中 / 低（0.9 / 0.5 / 0.15）
- 仅拖拽时显示
- 配置持久化

## 软件架构

### 1. 进程与线程模型

- 单进程，单 UI 消息循环
- 基于 `WM_TIMER` 的渲染调度
- 托盘菜单与渲染共享同一事件循环，减少线程切换开销

### 2. 渲染层

- 使用 `WS_EX_LAYERED + UpdateLayeredWindow` 实现透明叠加
- 使用 GDI+ 在 32 位 ARGB DIB 上绘制，再提交到分层窗口
- 仅使用小范围动态窗口（包围轨迹的最小区域），避免整屏重绘

### 3. 轨迹数据结构

- 固定长度环形缓冲区（300 槽位）
- 每个轨迹点包含：位置、尺寸、可见标记、颜色
- 每帧按衰减参数减少尺寸，实现拖尾消失效果

### 4. 调度与性能策略

- 自适应帧率
- 活跃态（移动/拖尾/按键变化）高频刷新
- 空闲态低频轮询
- 条件重绘：仅在视觉状态变化时提交分层窗口

### 5. 配置系统

- 配置文件路径：`文档\鼠标轨迹\config.txt`
- 配置文件为文本格式，使用 `|` 作为字段分隔符

## 构建与运行

### 环境要求

- Windows 11
- 已安装 WinLibs（`BrechtSanders.WinLibs.MCF.UCRT`）

### 构建

```powershell
cd zan
.\build.ps1
```

### 运行

```powershell
cd zan\build
.\shubiaoguiji.exe
```

启动后在系统托盘右键图标即可打开设置菜单。
